This is a web applciation where the backend is in NodeJS and database is in MongoDB. 
The frontend is created using ReactJS and is connected to the backend via HTTP. 

Steps to run the backend
1) Open Backend in Terminal
2) Type yarn install
3) Tyoe yarn start

Steps to run the frontend
1) Open Backend in Terminal
2) Type yarn install
3) Tyoe yarn start


The user needs to be logged in to access the pages. There are mainly 4 pages 
1) Home 
2) Jobs
3) About
4) Contact
